# Sociological Experiment Data Analyzer (SEDA) 🧪

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/yang280052-blip/soc-exp-analyzer/main/app.py)

A professional, easy-to-use tool for sociologists and students to analyze experiment data with advanced statistical methods.

## 🌟 Features

- **Data Cleaning**: Filter by response time or trap questions.
- **Descriptive Statistics**: Automatic normality checks and frequency tables.
- **Reliability & Validity**: Cronbach's Alpha, KMO, Bartlett, EFA, and CFA (AVE/CR).
- **Manipulation Checks**: T-tests and ANOVA with standard error plots.
- **Inferential Statistics**: ANCOVA and Multi-group Regression.
- **Advanced Effects**: Mediation (Hayes Model 4) and Moderation (Hayes Model 1) with Hierarchical Regression.

## 🚀 How to Use

1. **Upload Data**: Supported formats include CSV and Excel (xlsx/xls).
2. **Clean Data**: Use the module to filter out invalid responses.
3. **Select Analysis**: Choose from the sidebar to perform specific statistical tests.
4. **Interpret Results**: SEDA provides "Plain Language" academic interpretations for all results.

## 🛠️ Tech Stack

- **Streamlit**: For the interactive web interface.
- **Pingouin & Statsmodels**: For robust statistical calculations.
- **Semopy**: For Structural Equation Modeling and CFA.
- **Seaborn & Matplotlib**: For high-quality visualizations.
